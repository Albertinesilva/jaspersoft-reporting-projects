# 📊 Jaspersoft Reporting Projects

Repositório com projetos e experimentos utilizando **Jaspersoft Studio**, focado na criação de relatórios dinâmicos, modelagem de dados e geração de documentos para aplicações corporativas.

---

## 🚀 Objetivo

Este repositório tem como objetivo demonstrar na prática:

- Criação de relatórios com **Jaspersoft Studio**
- Integração com diferentes fontes de dados
- Organização e estruturação de templates de relatórios
- Exportação de relatórios em múltiplos formatos

---

## 🛠️ Tecnologias e Ferramentas

- Java
- Jaspersoft Studio
- JasperReports
- SQL (PostgreSQL / MySQL)
- JDBC

---

## 📚 Conceitos abordados

- 📄 Relatórios dinâmicos
- 📊 Tabelas, gráficos e sub-relatórios
- 🧩 Parâmetros e variáveis
- 🔗 Conexão com banco de dados
- 📦 Exportação para:
  - PDF
  - Excel (XLS/XLSX)
  - HTML

---

## 📂 Estrutura do Repositório

O repositório contém diferentes projetos e exemplos, incluindo:

- Templates `.jrxml`
- Relatórios compilados `.jasper`
- Conexões com banco de dados
- Exemplos práticos de uso

---

## 💡 Aplicações no mundo real

Os conhecimentos aplicados aqui são utilizados em:

- Sistemas corporativos (ERP)
- Sistemas governamentais
- Dashboards e relatórios gerenciais
- Geração de documentos automatizados

---

## ▶️ Como utilizar

1. Abra o projeto no **Jaspersoft Studio**
2. Configure a conexão com o banco de dados
3. Execute o relatório desejado
4. Exporte para o formato desejado
